A Lumi utiliza um sistema de reconhecimento de palavras-chave. Abaixo, listamos as principais perguntas que você pode fazer para testar as funcionalidades:


Tente estas frases:
Olá ou Oi
Quem é você?
Obrigado!

Perguntas de Vendas 
Valor/Preço:Qual o valor do curso? ou Quanto custa?
Garantia:O curso tem garantia? ou E se eu não gostar?
Pagamento:Posso pagar com Pix? ou Aceita cartão?
Certificado:Eu recebo certificado ao final?
Acesso:Por quanto tempo tenho acesso às aulas?
